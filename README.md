## Running Locally

Clone the project

```bash
git clone https://github.com/satnaing/simon.git
```

Go to the project directory

```bash
cd simon
```

Remove remote origin

```bash
git remote remove origin
```

Install dependencies

```bash
npm install
```

Start the server

```bash
npm start
```
